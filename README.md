# Personal Blog

My personal blog built with [Astro](https://astro.build/).

Based on [AstroPaper](https://github.com/satnaing/astro-paper) by [Sat Naing](https://satnaing.dev/), currently following the AstroPaper v5 structure:

- Astro 5 content collections in `src/content.config.ts` and `src/data/blog`
- Tailwind 4 through `@tailwindcss/vite`
- Pagefind search generated during `pnpm build`
